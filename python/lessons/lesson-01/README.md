# learning.python

## development environment

### pyenv config
```
$ pyenv install 3.10.4
$ pyenv local 3.10.4
```


### poetry config
```
$ poetry config --list
$ poetry config virtualenvs.in-project true 
```

### poetry init
```
$ poetry config --list
$ poetry config virtualenvs.in-project true 

$ poetry init -n
$ poetry install

$ poetry shell
$ deactivate / exit

$ poetry run python script.py
```

### poetry install
```
$ poetry add sqlalchemy 
$ poetry add pymysql
```
