## development environment

### pyenv config
```
$ pyenv install 3.10.5
$ pyenv local 3.10.5
```

### poetry config
```
$ poetry config --list
$ poetry config virtualenvs.in-project true 
```

### poetry init
```
$ poetry init -n
$ poetry install

$ poetry shell
$ deactivate / exit

$ poetry run python script.py
```

### poetry install package
```
$ poetry add sqlalchemy 
$ poetry add pymysql
```
